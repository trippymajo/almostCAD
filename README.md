# almostCAD
Simple Qt qmake project some kind of paint. C++ 14.

## Requirements
- g++ 12.2.0+
- cmake 3.1.0+
- Qt 5.XX

## Building
Linux:
```
cd ../almostCAD
mkdir BuilALL
cd BuilALL
cmake ../
cmake --build
```

### Figures:
1. Rectangle
1. Ellipse
